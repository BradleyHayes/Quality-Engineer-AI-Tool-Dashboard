# Aura AI Desktop Dashboard

This is a Python-based desktop version of the Aura AI Dashboard, converted from the original React web application.

## Prerequisites

1. **Python 3.8+** installed on your system.
2. **Ollama** (optional, for local AI): Download and install from [ollama.com](https://ollama.com).

## Installation

1. Install the required Python libraries:
   ```bash
   pip install PySimpleGUI ollama
   ```

2. Run the application:
   ```bash
   python aura_dashboard.py
   ```

## Features

- **Modular Layout**: Switch between Dashboard, AI Assistant, Task Organizer, and Project Hub.
- **Local AI Integration**: Uses Ollama to run models like Llama 3 locally on your machine.
- **Prompt Builder**: Advanced sidebar in AI Assistant with templates, auto-save, and history management.
- **Prompt History**: Save, load, and manage a library of your favorite prompts for quick access.
- **Custom Theme**: Matches the original turquoise and orange color scheme.
- **Responsive GUI**: Built with PySimpleGUI for a native desktop experience.
